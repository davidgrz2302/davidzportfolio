CineExplore

An iOS app that helps users discover popular movies and TV shows, view detailed info, and save favorites

Overview

CineExplore is a movie and TV show discovery app designed for iOS, built using Swift. Unlike streaming platforms, CineExplore doesn't provide playback—instead, it empowers users to explore thousands of titles and make informed decisions on what to watch. Users can search for movies and TV shows, browse detailed information, and save favorites for later—all within a clean, intuitive interface. Built for fans of film and TV, CineExplore focuses on exploration, personalization, and ease of use, making it perfect for both enthusiasts and casual viewers.

Achievements

  • Built native iOS application using Swift and UIKit, focused on fast performance and responsive design.
  • Integrated TMDB API to access thousands of movies and shows, with detailed metadata including overviews, ratings, cast, and genre.
  • Implemented Firebase Authentication with Google Sign-In to manage secure logins and persistent user profiles.
  • Created a Favorites system, allowing users to save and revisit their top picks from the movies and TV shows list.
  • Developed search functionality in both the Movies and TV Shows sections with real-time filtering and optimized API calls.
  • Designed a user-friendly Profile Page showing saved favorites and session data, with potential for future watch history tracking.
  • Applied clean UI/UX principles for accessibility across various user experience levels, from tech-savvy users to beginners.
