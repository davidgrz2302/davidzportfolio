USC GeoGuessr

A USC-themed recreation of GeoGuessr that challenges players to guess campus locations.

Overview

USC GeoGuessr is a location-based web game inspired by the popular GeoGuessr concept but tailored to the University of Southern California campus. Players are shown random images of locations around USC and must guess the exact spot on a map. The game supports both solo and multiplayer modes with real-time updates, accurate scoring based on proximity and response time, and user authentication. The app emphasizes engaging gameplay, smooth UI/UX, and robust backend integration.

Achievements

  • Developed full-stack multiplayer game using React (frontend), Spring Boot (backend), and MySQL (AWS Aurora RDS).
  • Implemented WebSocket communication for real-time multiplayer gameplay and state synchronization.
  • Built responsive map interface with Google Maps API integration, custom image-based location guessing, and dynamic scoring.
  • Enabled CI/CD with GitHub Actions for automatic deployment to GitHub Pages on push.
  • Conducted extensive black-box, unit, and stress testing for authentication, gameplay flow, reconnection, and leaderboard functionality.
  • Designed scalable backend architecture, including Spring Security, session management, and AWS S3 photo storage.
